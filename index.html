A Few Accomplishments

These are some example projects that I really enjoyed working at:

RoboCar

The goal of this project was to make a car that will perform a few tasks:
  1. Follow a '8' shapped black tape track (autonomous)
  2. Push a few boxes from a circle without leaving the circle (autonomous)
  3. Play football (app controlled)
How was that achieved:
  • the car was created using 3D printing and the desing was made using Tinkercad
  • the main component was Raspberry Pi that was coded using Python
  • the tasks were achieved using proximity and light sensors

Chess LED display

The purpose of this was to make a chess timer from an LED display that was coded using C#

Electric bike tracker

This IOT device was created using an antenna, a microcontroller, a GPS module and a Raspberry Pi. The GPS-enabled device creates a payload of data and transmits it through the antenna to a nearby LoRa Node, which is then transferred to the LoRa servers. The location can be viewed on a web page that will show the position of the bike in the last 24hours.
